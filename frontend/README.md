# PBALend Frontend

This is the web interface for the PBALend protocol.

## Prerequisites

1. Node.js installed.
2. Metamask extension installed.

## Setup

1. Install dependencies:

   ```bash
   npm install
   ```

2. Start the development server:

   ```bash
   npm run dev
   ```

3. Open [http://localhost:5173](http://localhost:5173) in your browser.

## Configuration

Update `src/App.jsx`:

- `CONTRACT_ADDRESS`: Set this to your deployed PBALend contract address.
